O Teste Rápido é a Melhor Escolha

Para a detecção de HIV, Sífilis e Hepatites B e C, os Testes Rápidos (TR) devem ser sempre a primeira opção. Eles são seguros, sigilosos e o resultado sai em minutos, permitindo o início imediato do tratamento e a quebra da cadeia de transmissão.

Guia de Testagem para Sífilis

Cenário 1: Diagnóstico inicial (sem histórico de sífilis)

A primeira escolha é sempre o Teste Rápido. Se o resultado for reagente, trate para sífilis (decida se trata para sífilis recente ou tardia) e solicite colete uma amostra de sangue para realizar o exame VDRL com o objetivo de confirmação diagnóstica.

Cenário 2: Suspeita de reinfecção (com histórico de sífilis)

O exame indicado é o VDRL. Ele ajuda a diferenciar uma nova infecção de uma cicatriz sorológica (memória da infecção anterior).

Cenário 3: Monitoramento do tratamento

Para acompanhar a eficácia do tratamento, o exame utilizado é o VDRL, realizado periodicamente conforme orientação médica.

Caso Especial: Gestantes

A testagem deve ocorrer no 1º, 2º e 3º trimestres. Se gestante sem histórico ou histórico desconhecido de sífilis, a preferência é pelo Teste Rápido. Se o resultado for reagente, trate para sífilis (decida se trata para sífilis recente ou tardia) e solicite colete uma amostra de sangue para realizar o exame VDRL com o objetivo de confirmação diagnóstica. Em caso de diagnóstico laboratorial, solicita-se VDRL e, mesmo que VDRL não reagente, laboratório fará automaticamente Teste Rápido (teste treponêmico) na mesma amostra. E se VDRL não reagente com teste rápido reagente, laboratório fará automaticamente terceiro teste com quimioluminescência (teste treponêmico) na mesma amostra com o objetivo de confirmação diagnóstrica.

Guia de Testagem para HIV

Diagnóstico

Sempre prefira o Teste Rápido. Com dois resultados reagentes em testes diferentes, o diagnóstico é confirmado na hora, agilizando o início do acompanhamento.

Quando solicitar o exame laboratorial?

O exame de laboratório (ANTI-HIV-1 + HIV-2) é indicado apenas se não for possível realizar o TR ou em casos de resultados divergentes nos testes rápidos.

Guia de Testagem para Hepatites B e C

Diagnóstico

A recomendação principal para as hepatites virais B e C também é o Teste Rápido, pela sua agilidade e eficácia.

Quando solicitar o exame laboratorial?

Solicite o exame laboratorial (HBsAg para Hepatite B e Anti-HCV para Hepatite C) somente quando a realização do Teste Rápido não for possível.
